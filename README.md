# PaceMate Website

Professional landing page website for PaceMate - Energy Tracking for Chronic Illness.

## Files

- `index.html` - Landing page with features and call-to-action
- `privacy.html` - Privacy Policy
- `terms.html` - Terms of Service
- `contact.html` - Support and FAQ page

## Deployment Options

### Option 1: GitHub Pages (Free)

1. Create a new GitHub repository named `pacemate-website`
2. Push these files to the repository
3. Go to Settings → Pages
4. Select "Deploy from a branch"
5. Select `main` branch and `/root` folder
6. Your site will be available at `https://yourusername.github.io/pacemate-website`
7. Set up a custom domain (optional)

### Option 2: Netlify (Free)

1. Go to [Netlify](https://netlify.com)
2. Sign up with GitHub
3. Click "New site from Git"
4. Select your repository
5. Deploy settings:
   - Build command: (leave blank)
   - Publish directory: `.` (current directory)
6. Your site will be live in seconds
7. Set up custom domain in Settings

### Option 3: Vercel (Free)

1. Go to [Vercel](https://vercel.com)
2. Sign up with GitHub
3. Import your repository
4. Deploy (no build steps needed)
5. Your site will be live
6. Set up custom domain in Settings

### Option 4: Traditional Hosting

Upload all HTML files to your web hosting provider via FTP or file manager.

## Custom Domain Setup

To use `https://pacemate.app`:

1. Register domain at [Namecheap](https://namecheap.com), [GoDaddy](https://godaddy.com), or similar
2. Point domain to your hosting provider
3. Add SSL certificate (most providers include free SSL)
4. Update links in app store listings to match

## Important Links to Update

Before deploying, update these placeholders:

1. **App Store Links** (in `index.html` and `contact.html`)
   - Replace `https://apps.apple.com/app/pacemate` with actual iOS App Store link
   - Replace `https://play.google.com/store/apps/details?id=space.manus.pacemate` with actual Google Play link

2. **Email Address** (in `contact.html`)
   - `support@pacemate.app` → Your support email

3. **Website URLs** (throughout all files)
   - `https://pacemate.app` → Your actual domain
